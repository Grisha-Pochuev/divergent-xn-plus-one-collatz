# Contributing

Contributions are welcome, especially independently checkable lemmas, small verification scripts, and precise counterexamples to proposed arguments.

Please distinguish clearly between:

- **proved**: accompanied by a complete argument or finite certificate;
- **verified for a finite range**: accompanied by exact parameters and code;
- **heuristic**: probabilistic or experimental motivation;
- **conjectural**: an unproved proposed statement.

A proposed divergence proof should include an independently runnable checker whenever computation is used. Long trajectory output alone is not sufficient.
