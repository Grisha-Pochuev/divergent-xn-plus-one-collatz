# Simultaneous finite barriers for cycle length and cycle height

## Theorem

For every pair of positive integers `(B,H)`, one can explicitly construct an odd multiplier `Q>=5` such that the accelerated orbit from `n0=1`

1. never returns to `1`;
2. cannot enter a nontrivial positive cycle of length at most `B`;
3. cannot enter a nontrivial positive cycle containing any value at most `H`.

Equivalently, if this orbit is periodic, then its cycle has length greater than `B` and every cycle element is greater than `H`.

The multiplier depends on `(B,H)`, so this remains a finite-barrier theorem rather than a final divergent orbit.

## 1. A Mersenne divisor creates a height barrier

Choose an odd multiple `m` of `3` such that

```text
M = 2^m-1 > H.
```

Because `3|m`, the Mersenne number `M` is divisible by `7`. Because `m` is odd, `M` is not divisible by `3`.

We will choose `Q` divisible by `3M` and divisible by `3` exactly once.

For every accelerated output `y=C_Q(n)` with halving count `a`,

```text
2^a*y == 1  (mod M),
```

so

```text
y == 2^(-a)  (mod M).
```

The order of `2` modulo `M=2^m-1` is exactly `m`; hence the least nonnegative residues in the subgroup generated by `2` are

```text
1,2,4,...,2^(m-1).
```

If the residue is `2^b` with `b>=1`, it is even. Since `y` is odd, the least possible positive representative is

```text
M+2^b >= M+2.
```

If the residue is `1`, oddness permits `y=1`, but the next possible odd representative is `1+2M`.

Therefore every accelerated output other than `1` satisfies

```text
y >= M+2 > H.                                    (1)
```

## 2. Exact construction near a half-integral power of two

Choose `K` so large that

```text
2^K > 40*B*M.                                    (2)
```

Put

```text
R = 2^K*sqrt(2).
```

Choose `Q` to be the smallest integer larger than `R` of the form

```text
Q = 3*M*t,
```

where `t` is odd and not divisible by `3`.

Allowed `t` values are `1` and `5` modulo `6`, so the gap between successive allowed values is at most `4`. Hence

```text
0 < Q-R <= 12*M.                                 (3)
```

The construction uses only an integer square root and finite congruence checks.

## 3. Permanent avoidance of 1

The multiplier `Q` is divisible by `3` exactly once and by `7`. If

```text
ell=ord_Q(2),
```

then `ell` is divisible by both `ord_3(2)=2` and `ord_7(2)=3`, hence by `6`. The lifting-the-exponent lemma gives

```text
v3(2^ell-1)>=2>v3(Q).
```

Thus

```text
gcd(Q,(2^ell-1)/Q)>=3.
```

Every direct predecessor of `1` shares a factor with `Q`, while every accelerated output is coprime to `Q`. Also, by (2)-(3), `Q+1` lies strictly between consecutive powers `2^K` and `2^(K+1)`, so the first step is not `1`.

The orbit from `1` therefore never returns to `1`. Combining this with (1), every later orbit value is greater than `H`.

## 4. Length barrier

Suppose the orbit entered a cycle of length `p<=B`. Let `A` be the total halving count around the cycle. By the exact product identity and (1),

```text
Q^p < 2^A <= (Q+1/(M+2))^p.                     (4)
```

Write

```text
U = Q+1/(M+2) = R+e.
```

From (3), for `M>=7`,

```text
e < 12*M+1 < 13*M.
```

Using (2),

```text
p*e/R < 13*B*M/2^K < 13/40 < 1/3.
```

Consequently,

```text
(U/R)^p < exp(1/3) < sqrt(2).                    (5)
```

If `p` is even, `R^p` is an integer power of two and (4)-(5) place `2^A` strictly between that power and the next one.

If `p` is odd, `R^p` is a half-integral power of two and (4)-(5) place `2^A` strictly below the next integer power.

Both are impossible. Hence `p>B`.

## 5. Meaning

This theorem shows that finite evidence can be strengthened in two independent directions without trajectory simulation: a hypothetical cycle can be forced to be both arbitrarily long and arbitrarily high.

The remaining infinite gap is explicit: `Q` changes when either requested barrier changes. A complete prize solution still requires one fixed multiplier for which all finite barriers hold simultaneously.
