# Wieferich adjacent-label coordinates modulo `q^2`

Let `q` be an odd prime and put

```text
h=ord_q(2).
```

Assume the Wieferich condition

```text
2^h == 1 (mod q^2).                                 (1)
```

Let `X` be an odd multiplier satisfying

```text
X=q*c,
q does not divide c.
```

Consider any positive accelerated orbit

```text
2^a_i*n_(i+1)=X*n_i+1.                              (2)
```

## 1. Least full labels

For every transition define `s_i` to be the unique integer

```text
1<=s_i<=h,
a_i == s_i (mod h).
```

Reducing (2) modulo `q` gives

```text
n_(i+1) == 2^(-s_i) (mod q).                        (3)
```

Thus every accelerated output lies in the subgroup generated by `2` modulo
`q`.

## 2. The full `q^2` coordinate

Write

```text
a_i=s_i+h*k_i.
```

By (1),

```text
2^a_i == 2^s_i (mod q^2),
```

independently of the layer `k_i`.  Therefore (2) gives

```text
2^s_i*n_(i+1) == 1+q*c*n_i (mod q^2).               (4)
```

Using the previous transition in (3),

```text
n_i == 2^(-s_(i-1)) (mod q).
```

Substitution into (4) yields the exact adjacent-label formula

```text
n_(i+1)
 ==2^(-s_i)*(1+q*c*2^(-s_(i-1))) (mod q^2).         (5)
```

Hence the residue of every value with two preceding orbit transitions is
completely determined by the ordered pair

```text
(s_(i-1),s_i).
```

No unknown full-order layer occurs in (5).

## 3. Exact class count

The map from ordered label pairs to residues modulo `q^2` is injective.

Indeed, reduction modulo `q` recovers `s_i` from

```text
n_(i+1)==2^(-s_i) (mod q).
```

After `s_i` is known, division of

```text
2^s_i*n_(i+1)-1
```

by `q`, followed by multiplication by `c^(-1)` modulo `q`, recovers
`2^(-s_(i-1))` and therefore `s_(i-1)`.

Consequently exactly

```text
h^2
```

residue classes modulo `q^2` can occur after two transitions.

For

```text
q=1093,
h=364,
```

this is

```text
364^2=132496 classes.                                (6)
```

By comparison, the one-step output condition alone allows

```text
h*q=397852
```

classes modulo `q^2`.  The adjacent-label theorem permanently removes exactly
two thirds of those lifts.

## 4. Application to the hybrid candidates

The hypotheses hold for both

```text
X=2^156-9,   c==151 (mod 1093),
X=2^260-3,   c==936 (mod 1093).
```

Thus every value sufficiently far along either candidate orbit, and every value
of a hypothetical positive cycle, lies in one of exactly `132496` explicit
classes modulo

```text
1093^2=1194649.
```

This is an infinite-family sieve, not a trajectory cutoff.  It does not by
itself exclude cycles, because each surviving residue class has infinitely many
positive representatives.  Its intended use is in a harmonic-packing or
height-credit argument beyond the current finite cycle barriers.
